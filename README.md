# ADLproject - tinyIMGNET - classification
Advenced Deeplearning Project2 - Using tiny IMGNET dataset

## Introduction
- The goal of this project is to achieve high accuracy on Tiny-IMGNET image classification.
- This project implements a full deep learning pipeline, inlcuding data preprocessing, data augmentation, model definition, and training loop.
